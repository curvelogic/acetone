04 / 08

Decisions taken · for ratification

Four ADRs the fixes forced.

ADR-0028

Referential integrity at the transaction boundary

One check, in save_in_place, that no writer — import, Cypher, merge completion — can leave an edge without its endpoints. The naive dangling-merge became structurally impossible.

ADR-0029

Preserve deferred property types on write-back

Targeted fix for U2; the full typed value channel is Phase-7 (vdc). An unchanged Bytes/temporal property keeps its stored value verbatim.

ADR-0030

No parallel edges in v0.1

CREATE of a duplicate edge is rejected, not silently collapsed. The format reserves the discriminator, so query-reachable edge identity (rid, Phase 7) needs no migration.

ADR-0031

Pre-0.1 spec-drift corrections

Removed phantom push/pull/clone and :diff; corrected §5.3 to the materialised pipeline; documented SHA-1 as the deliberate default; sharpened the 0.1 positioning.

Also queued for the boundary: ratify ADR-0023–0027 (Phase 6 + composite index keys), and the two Gate-D-flagged decisions.
